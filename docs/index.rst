Welcome to Bottery's documentation!
===================================

Welcome to Bottery!

Bottery is a framework made for building bots on multiple platforms (such as 
Telegram, Facebook Messenger, etc...).

We are still under construction, so any feedback is welcome.

To install use::
    $ pip install bottery
